# Altiora full working website

This is a complete Vite + React + Tailwind project.

## Install
```bash
npm install
```

## Run locally
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Files included
- full React site
- Tailwind config
- Vite config
- PostCSS config
- Altiora SVG logo
- bilingual EN/FR switch
- all requested latest edits
